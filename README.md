# 🏡 Family Members World – Interactive English Learning App v2.0

![Ages](https://img.shields.io/badge/Ages-7--10-green) ![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Ready-orange) ![Games](https://img.shields.io/badge/Games-7-purple)

A rich, interactive educational web app for children ages 7–10. Features **7 unique games**, flashcards, quiz, practice, and a progress system with badges.

---

## 🎮 Activities (10 Total)

| # | Activity | Description |
|---|---|---|
| 1 | 📖 Learn | Vocab cards + 2 YouTube videos |
| 2 | 🃏 Flashcards | Flip cards with TTS pronunciation |
| 3 | 🎱 Bingo | 5×5 bingo card with word calling |
| 4 | 🔀 Unscramble | Tap letters to spell words |
| 5 | 🕵️ Guess Who? | Multi-clue riddle game |
| 6 | 🎈 Balloon Pop | Pop balloons matching target word |
| 7 | 🏗️ Sentence Builder | Drag word blocks to build sentences |
| 8 | 🎡 Family Wheel | Spin wheel → answer question |
| 9 | 🏃 Race to House | Answer correctly to move the runner |
| 10 | 🧠 Final Quiz | 10-question quiz with scoring |

## 🚀 Run Locally

Just open `index.html` in any modern browser. No server needed.

## 📤 Upload to GitHub

```bash
git init
git add .
git commit -m "Family Members World v2"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/family-members.git
git push -u origin main
```

## 🌐 Publish on GitHub Pages

1. Go to repo **Settings → Pages**
2. Source: **Deploy from branch → main → / (root)**
3. Save → visit `https://YOUR_USERNAME.github.io/family-members/`

## ✨ Features

- 🎮 **7 unique interactive games** including Bingo, Balloon Pop, Wheel & Race
- 🔊 Text-to-speech pronunciation (Web Speech API)
- 💾 Progress saved with `localStorage`
- 📱 Fully responsive (mobile, tablet, desktop)
- 🏆 Scoring system with badges and stars
- 🌐 Bilingual support (English + Spanish hints)
- 🎨 Dark hero design with colorful game cards
- 🌀 Animated spinning wheel rendered on Canvas

---

## 🛠️ Tech Stack

HTML5 · CSS3 · Vanilla JavaScript · Web Speech API · Canvas API · localStorage
